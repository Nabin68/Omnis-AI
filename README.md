# 🌌 Omnis AI

**Omnis** – Latin for *"all"* – is not just another assistant.  
It's an **all-in-one AI powerhouse** that can automate, interact, and evolve with you.

From controlling your system to managing your social media, coding, generating images, and even making WhatsApp calls — Omnis is designed to do **literally everything.**

---

## ✨ Key Features

- 🎙️ **Voice Assistant** – Talk to Omnis in real-time
- 🖥️ **System Automation** – Control your entire system with voice commands
- 📺 **Social Media Automation** – Automate **YouTube, Instagram, Facebook**, and more
- 📈 **Finance & Share Market** – Fetch live stock data, analyze trends, and automate trading tasks
- 🎟️ **Ticket Booking** – Check and book tickets in real-time
- 📡 **Realtime Intelligence** – Get the latest news, weather, and trending topics
- 💻 **Code Generation** – Write and execute code across multiple languages
- 📧 **Email Automation** – Draft and send emails (and even send your code)
- 🖼️ **Image Generation** – Create AI-powered art and visuals
- 💬 **WhatsApp Automation** – Send texts and even make phone calls directly from Omnis
- 🎨 **Cool GUI** *(coming soon)* – A futuristic interface for seamless interaction
- 🗂️ **Memory & Data Storage** – Keeps past chats, generated images, and contextual history

**Omnis isn't just an assistant.**  
**It's the future of personal AI.**

## 🚀 Installation

### 1. Clone the repository
```bash
git clone https://github.com/Nabin68/omnis-ai.git
cd omnis-ai
```

### 2. Create & activate virtual environment
```bash
# macOS/Linux
python -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Set up environment variables
Create a `.env` file in the root directory:
```env
API_KEY=your_api_key_here
OTHER_SECRET=your_secret_here
```

### 5. Run Omnis
```bash
python main.py
```
